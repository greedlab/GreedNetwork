# GreedEmoji

Convert apple emoji to aliased string or aliased string to apple emoji

## Installation

pod 'GreedEmoji'

## Usage

```
#import "NSString+GreedEmoji.h"

    // emoji to aliased string
    [@":angel:" aliasedString];
    // aliased string to emoji
    [@"\U00002693" emojizedString];
```
